# Job Evaluation Framework

## Scoring Dimensions

### 1. Technical Skills Match (0-100)
| Score | Meaning |
|-------|---------|
| 80-100 | Core requirements are primary skills |
| 60-79 | Most requirements match, 1-2 learnable gaps |
| 40-59 | Partial match, significant upskilling needed |
| 0-39 | Fundamental mismatch |

### 2. Experience Match (0-100)
| Score | Meaning |
|-------|---------|
| 80-100 | Direct experience in same domain and role type |
| 60-79 | Related experience, transferable skills clear |
| 40-59 | Adjacent experience, would need to make the case |
| 0-39 | Unrelated experience |

### 3. Behavioral/Culture Fit (0-100)
| Score | Meaning |
|-------|---------|
| 80-100 | Culture strongly matches behavioral preferences |
| 60-79 | Mixed signals but mostly compatible |
| 40-59 | Some friction areas |
| 0-39 | Significant culture mismatch |

### 4. Location & Logistics (Pass/Fail)
- Within commute range: PASS
- Remote with occasional office: PASS
- Requires relocation: FAIL
- Frequent international travel: FLAG

### 5. Career Alignment & Motivation (0-100)
| Score | Meaning |
|-------|---------|
| 80-100 | Strongly aligned, clear growth path |
| 60-79 | Good role but partially aligned |
| 40-59 | Decent but doesn't build toward goals |
| 0-39 | Dead end or backwards step |

## Weighting
- Technical Skills: 30%
- Experience Match: 25%
- Behavioral Fit: 15%
- Career Alignment: 30%
- Location: pass/fail (not weighted)

## Thresholds
- **Strong Fit** (75+): Definitely apply, tailor everything
- **Good Fit** (60-74): Apply, address gaps in cover letter
- **Moderate Fit** (45-59): Consider carefully
- **Weak Fit** (30-44): Probably skip
- **Poor Fit** (<30): Skip
